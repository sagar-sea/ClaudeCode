# GenAI Learning Reference

An interactive HTML reference for Generative AI concepts and terminologies.

## Features

- **Categorized Cards**: Organized by concept categories (Fundamental, Technical, Tools, Prompt Engineering)
- **Three Viewing Modes**: Simple description, Technical description, and Practical examples
- **Search Functionality**: Find concepts quickly using the search box
- **Category Filtering**: Filter by specific categories
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. **Open the File**: Open `index.html` in your web browser
2. **Browse Cards**: Scroll through the concept cards
3. **Switch Views**: Click on the tabs (Simple, Technical, Examples) to see different information for each concept
4. **Filter by Category**: Use the category buttons at the top
5. **Search**: Use the search box to find specific terms

## Concepts Covered

### Artificial Intelligence (AI)
The foundation of all intelligent computer systems

### Large Language Model (LLM)
AI models trained on massive text data for language understanding and generation

### Parameters
The "memory" units that store learned patterns in AI models

### Prompt Engineering
Techniques for writing effective prompts to get better AI responses

### Model Context Protocol (MCP)
Standard for AI models to safely interact with external tools

### Skills
Specialized capabilities that extend AI functionality

### Tokens
Basic units of text processing in AI models

### Fine-tuning
Adapting pre-trained models for specific tasks

### RAG (Retrieval-Augmented Generation)
Combining external knowledge retrieval with text generation

### Zero-shot Learning
AI performing tasks without specific training examples

### Few-shot Learning
Providing a few examples to guide AI behavior

### Hallucination
AI generating plausible but incorrect information

### Transformer Architecture
The neural network design powering modern AI

### Temperature
Parameter controlling randomness in AI responses

### Ollama
Framework for running AI models locally
